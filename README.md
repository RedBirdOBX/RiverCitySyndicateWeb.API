# RiverCitySyndicate.API

## Concept

API provides Shows, Videos, and Photos data for web app. Has MS Sql Server backend database. Also uses Serilog for Console, log file, and data table logging.

## Packages 

Installed Custom Packages:
- AutoMapper  
- AutoMapper.Extensions.Microsoft.DependencyInjection
- Serilog
- Serilog.AspNetCore
- Serilog.Sinks.Console
- Serilog.Sinks.File
- Serilog.Sinks.MSSqlServer
- Serilog.Sinks.PeriodicBatching
